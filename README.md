## 你将在这里学习到的技能

- git 版本管理
- atom 代码编辑器
- shell 脚本
- es6/7 新语法和API的使用
- babel 编译器及其 plugins
- react & react-dom
- webpack 2.2.1 & webpack-dev-server
- yarn & npm 包管理器
- react-router 路由管理
- axios 基于Promise的HTTP库
- CSS3 & flex layout 现代化布局模式
- 架构思想和组件化思维
